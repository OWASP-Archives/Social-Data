== Summary ==

The purpose of this initiative is to set a default training structure for local chapters to use when offer OWASP training to OWASP members.

== Training Topics ==

*OWASP Tour - The purpose of this training is to provide an introduction to OWASP.
*OWASP Top Ten - The purpose of this training is to deliver the latest top 10 common vulnerabilities found in web applications
*OWASP Testing Guide
*OWASP Software Assurance Maturity Model 
*OWASP Code Review Guide
