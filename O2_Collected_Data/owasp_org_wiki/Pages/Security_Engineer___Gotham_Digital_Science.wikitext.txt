'''Company: Gotham Digital Science

Job Title & Location: 

Security Engineer - New York, NY, USA

Job Description:'''

Gotham Digital Science is looking for a smart, energetic and motivated individual to add to its elite team of ethical hackers and consultants.  As a security engineer on our team, you will: 

•	Perform application security assessments against custom software applications 

•	Validate and prioritize the results produced by various security tools 

•	Help clients integrate commercial and open source security tools into their development environments, build processes, and overall Software Development Life Cycle

•	Work closely with developers and corporate security teams to identify and remediate security issues

•	Train developers and engineers on how to use various commercial and open-source software security tools
GDS provides a challenging and exciting work environment that offers a healthy combination of autonomy and senior level support.  Our team publishes books and security blogs, contributes to open source software projects, and are engaged in a variety of continuous security research projects.  If you enjoy performing deep technical work in a fun and casual atmosphere, contact us to find out more about joining our team.  

'''Qualifications:'''

The following qualifications are expected from potential applicants: 

•	University degree from an accredited college or university

•	Familiarity with application and network security concepts 

•	Excellent teaming and communication skills 

The following skills are not required from applicants but would be considered a plus: 

•	Experience running commercial/open-source secure application development tools/products, such as Fortify Source Code Analyzer or HP WebInspect

•	University and/or Graduate Degree in Computer Science, Information Systems, Engineering or related major 

•	Development and/or source code review experience in C/C++, C#, VB.NET, or Java

•	Penetration testing experience above and beyond running automated tools
