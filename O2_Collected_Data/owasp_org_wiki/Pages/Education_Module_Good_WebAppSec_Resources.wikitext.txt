= Module Description =
This module points you to some good web application security resources when developping or deploying web applications as part of the [[:Category:OWASP Education Project|Education Project]].
OWASP education tracks are only the beginning of your journey. Web application security is a moving target. New vulnerabilities and threats are discovered regularly. Web application security controls are becoming mature. The following resources should provide you with enough pointers to serve both as reference and for further research. 
* Hard Copy 
* Web Sites 
* Mailing lists 
* Blogs 

= Target audience =
Novice.
