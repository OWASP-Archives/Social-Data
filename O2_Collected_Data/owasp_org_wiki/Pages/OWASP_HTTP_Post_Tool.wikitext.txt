The OWASP Testing Guide v3 [http://www.owasp.org/index.php/Testing_for_Denial_of_Service (Section 4.9) Denial of Service Testing] outlines numerous areas of Denial of Service. This QA tool was created to allow you to test your web applications to test availability concerns from Layer7 DoS HTTP GET and HTTP POST denial of service attacks. <br>
