<B>OWASP AppSec India Conference 2008</B> happened to be first ever biggest application security event in India. Event was organized at India Habitat Center, Lodhi Road, New Delhi. It was two days event with first day focussed at talks on various burning application security topics and business case studies. Second day, there were 6 training sessions, those were conducted by most reputed, renowned application security gurus across the world Like Nish Bhalla, Jason Li, Shreeraj Shah, Mano Paul, etc. There were more than 300 participants at the event with HP Software being Diamond Sponsor to this biggest ever Application Security Event in India.
