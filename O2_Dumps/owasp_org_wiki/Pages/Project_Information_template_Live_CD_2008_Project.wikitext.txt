 | style="width:15%; background:#7B8ABD" align="center"| '''Short Project Description''' 
 | colspan="6" style="width:85%; background:#cccccc" align="left"|The OWASP Live CD is a project that collects some of the best open source security projects in a single environment.  Web developers, testers and security professionals can boot from this Live CD and have access to a full security testing suite. This allows its users to test for various security issues in web applications and web sites. The OWASP Live CD also contains documentation and an interactive learning environment ([[:Category:OWASP_WebGoat_Project|Web Goat]]) to enhance users web application security knowledge.  This presentation will cover the current state of the OWASP Live CD as well as the plans for future developments.  The OWASP Live CD is free for commercial or nonÂ­commercial use.
