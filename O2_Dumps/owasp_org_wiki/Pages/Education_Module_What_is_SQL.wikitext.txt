= Module Description =
This module is an example module covering 'What is SQL' as part of the [[:Category:OWASP Education Project|Education Project]] based on the [[Education Module Sample]]. This module is to explain SQL to non-programmers. This can be used as support prior to the [[Education Module What Is SQL Injection|What is SQL Injection?]] module.

= Target audience =
Novice.

The presentation can be found in [[:Image:Module_What_is_SQL.zip|What is SQL]].
