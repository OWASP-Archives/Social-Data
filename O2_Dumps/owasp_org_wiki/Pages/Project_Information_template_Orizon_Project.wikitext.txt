| style="width:15%; background:#7B8ABD" align="center" | '''Short Project Description''' 
| colspan="7" style="width:85%; background:#cccccc" align="left" | This project born in 2006 in order to provide a framework to all Owasp projects developing code review services. The project is in a quite stable stage and it is usable for Java static code review and some dynamic tests against XSS. Owasp Orizon includes also APIs for code crawling, usable for code crawling tools.
