= Module Description =
This module explains some good secure development practices when developping or deploying web applications as part of the [[:Category:OWASP Education Project|Education Project]].
Next to the Top 10 remedies this module provides some good secure development practices from the 
* OWASP Guide, covering e.g.
* Validating User Input 
* Authentication
* Authorization
* Session Management
* Using Interpreters
* Crypto
* Catching Errors
* File System
* Configuration
* Web 2.0

= Target audience =
Novice.

= Presentation =
The presentation can be found in [[:Image:Education_Module_Good_Secure_Development_Practices.ppt|Good Secure Development Practices]].

Normally this presentation is performed in 70 minutes.
